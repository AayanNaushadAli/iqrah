# Quran PDF Files

Place your Quran chapter PDF files in this directory with the following naming convention:

## File Naming Pattern
`chapter-{number}.pdf`

### Examples:
- `chapter-1.pdf` - Al-Fatiha (Chapter 1)
- `chapter-2.pdf` - Al-Baqarah (Chapter 2)
- `chapter-3.pdf` - Al-Imran (Chapter 3)
- ...
- `chapter-114.pdf` - An-Nas (Chapter 114)

## How to Add PDFs

1. Obtain PDF files for each chapter of the Quran
2. Rename them according to the pattern above
3. Place them in this `pdfs/` directory
4. Open the app in your browser
5. Click on any Surah from the menu to view its PDF

## Total Files Needed
You need **114 PDF files** (one for each chapter of the Quran)

## Features
- Click any Surah name to view its PDF
- Download button to save each chapter
- File info shows chapter name and verse count
- Arrow keys navigate between chapters (Left/Right arrows)
- Responsive design works on all devices

## Notes
- PDFs will be displayed in an embedded viewer
- Make sure PDF file names exactly match the pattern
- Each file should contain the chapter content you want displayed
